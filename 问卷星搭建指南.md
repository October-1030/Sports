### 📄 第4页：身体素质评估

#### Q11. 🏃‍♂️ 短距离冲刺测试 *
- **题型：** 单选
- **题目描述：**
```
💡 测试方法：
在家中走廊或小区空地，让孩子全力冲刺约10米距离。
观察与同龄孩子相比的速度表现。
```
- **选项：**
  - 🥇 明显比同龄孩子快（反应迅速，启动快，步频高）
  - 🥈 比大部分同龄孩子快（跑起来比较轻松）
  - 🥉 和大部分同龄孩子差不多（中等水平）
  - 📝 比大部分同龄孩子慢（跑起来比较吃力）

#### Q12. 🦘 立定跳远测试 *
- **题型：** 单选
- **题目描述：**
```
💡 测试方法：
让孩子双脚并拢站立，原地用力向前跳。
可用卷尺测量距离，或用地砖/A4纸（约30cm）作为参照。
```
- **选项：**
  - 🥇 能跳过1米以上（约3-4张A4纸长度）
  - 🥈 能跳过80-100厘米（约2.5-3张A4纸）
  - 🥉 能跳过60-80厘米（约2-2.5张A4纸）
  - 📝 跳不过60厘米（少于2张A4纸）

#### Q13. 🤹‍♂️ 单脚平衡测试 *
- **题型：** 单选
- **题目描述：**
```
💡 测试方法：
让孩子单脚站立（另一只脚抬起），双手叉腰或伸展保持平衡。
观察能坚持多久不晃动。
```
- **选项：**
  - 🥇 能稳定站立30秒以上（闭眼也能站10秒+）
  - 🥈 能站立15-30秒（身体偶尔晃动）
  - 🥉 能站立5-15秒（需要调整姿势）
  - 📝 站不到5秒（很快失去平衡）

#### Q14. 🧘 柔韧性测试（坐位体前屈）*
- **题型：** 单选
- **题目描述：**
```
💡 测试方法：
让孩子坐在地上，双腿伸直并拢，脚尖朝上。
身体前倾，双手慢慢向前伸，尝试触碰脚趾。
```
- **选项：**
  - 🥇 手掌能完全超过脚趾5厘米以上（柔韧性优秀）
  - 🥈 手指能触碰到脚趾或略微超过（柔韧性良好）
  - 🥉 手指距离脚趾5-10厘米（柔韧性一般）
  - 📝 手指距离脚趾超过10厘米（需要加强柔韧性训练）

### 📄 第6页：心理特征

#### Q16. 🎯 运动技能学习速度观察 *
- **题型：** 单选
- **题目描述：**
```
💡 观察重点：
回忆孩子学习新运动技能（如骑车、跳绳、拍球、游泳）时的表现。
```
- **选项：**
  - 🚀 很快掌握（通常几次示范就能模仿，不到一周就能熟练）
  - 📚 正常速度（需要1-2周反复练习才能掌握基本动作）
  - 🐢 学得较慢（需要几周甚至更长时间，需要大量耐心指导）

#### Q17. 💪 面对困难和失败时的反应 *
- **题型：** 单选
- **题目描述：**
```
💡 场景回忆：
想想孩子在运动中遇到困难时（如连续投篮不中、跑步落后、
学不会某个动作）的表现。
```
- **选项：**
  - 🔥 越挫越勇（失败后更努力，主动要求再试一次）
  - 🤗 需要鼓励（失败后情绪低落，需要大人安慰和鼓励才能继续）
  - 😔 容易放弃（遇到困难就说"我不想玩了"、"太难了"）
